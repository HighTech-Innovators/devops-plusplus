# Glossary

- **IaC (Infrastructure as Code)**: Managing infrastructure through code rather than manual processes.
- **OPUA (Operations Per User Action)**: A DevOps++ metric that measures efficiency in terms of system work per user action.
- **RUM (Real User Monitoring)**: Monitoring performance based on actual user interactions.
- **Wasm (WebAssembly)**: A lightweight execution format for running code securely and efficiently.
